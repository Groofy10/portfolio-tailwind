About Me

Halo, saya Ferdy Jonathan Angwijaya! Saat ini, saya adalah mahasiswa di Binus University, mengambil jurusan Computer Science. Saya sedang belajar web development dan sangat tertarik dengan dunia pengembangan perangkat lunak, terutama di bagian backend.

Saat ini, saya sedang fokus belajar menggunakan framework Laravel dan Express.js untuk pengembangan web di bagian backend. Saya akan terus berusaha mengembangkan keterampilan saya di kedua framework tersebut agar semakin terbiasa dan mahir menggunakannya. Selain itu, saya juga akan terus berusaha untuk berkembang dengan mempelajari framework dan bahasa pemrograman lain dan juga belajar di bagian frontend selain dari html dan css dasar seperti belajar framework react dan lainnya.
